Algorithms and Data Structures sample codes in Java 
Collection of samples implementations of common algorithms (Sorting and Searching) and Data structures 
in Java. 
These sample codes include: 
Searching Algorithms:
- Linear 
- Binary
Sorting Algorithms:
- Bubble Sort 
- Bucket Sort
- Counting Sort 
- Inserting Sort etc 

Data Structures: 
- LinkedList 
- Doubly LinkedList 
- Queue
- HashTable 
- Stack etc
